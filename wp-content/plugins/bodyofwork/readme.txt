Body of Work creates a custom post type for artists to manage their portfolio. 

== Description ==

Creates a bustom post type called 'Work' that is for storing and categorizing artwork. In combination with other plugins, this can be used to create a portfolio site that keeps the work separate from the blog.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
